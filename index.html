Operations in the Presentation

  • → ↓ Click on the right half : Next Slide
  • ← ↑ Click on the left half : Previous Slide
  • 1 Shift+← : Show the First Slide
  • E Shift+→ : Show the Last Slide
  • R Return : Reload Current Slide ("Panic button" for WHITE screen)
  • F : Full Screen Mode
  • esc : Exit Full Screen Mode

How to Make Your Presentation

Short Story

Place the “contents.js” file on the same folder of this "index.html" file. Open the "index.htm" and click link!

How to Prepare the “contents.js”

A template of “contents.js” is the file “seed-contents.js” located on the “projector” folder. Just copy it onto the right place and edit it. The name of the file have to change to “contents.js” from “seed-contents.js.”

The contents of the “seed-contents.js” are referring some images, and they have to be in the “images” folder as the same directory of the file “contents.js.” The “seed-images” folder have to be copied and renamed to “images.”
